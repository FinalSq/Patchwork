Patchwork: A Coding Project

Piece it together with Patchwork! A javascript coding exercise using loops. Write a program that prints the numbers from 1 to 100 and for multiples of 3 display "Patch" instead of the number. For multiples of 5 display "work". If the number is a multiple of both 3 & 5, display "Patchwork".

Instructions

Enter a starting value in the first input box. Then enter a value in the second input box. Press the button and watch the magic happen.
